# Repository Agent Instructions

## Line Endings

- Use LF line endings for all text files in this repository.
- Keep `.gitattributes` as the source of truth for line-ending normalization.
- Do not introduce mixed line endings in edited files.
- Before committing broad file edits, verify tracked text files do not contain CRLF line endings.

## Workflow

- Keep changes small and focused.
- Work on a feature branch rather than committing directly to `development`, `main`, or `master`.
- Preserve unrelated user changes.
- Run the most relevant verification before committing or opening a PR.
